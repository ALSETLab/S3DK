OSS

For over 10 years, (Luigi Vanfretti) has heard the expression "What can you do with real-time PMU data?", for which many answers can be given. However, any answer that you might provide assummes that you have the technology to actually make it a reality, which is not the case for most researchers. S3DK changes that, and it makes the answer, "as long as you can program (in C++ or LabView...), its up to you what application to come up with!.
